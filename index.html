Descobrir Idade
<...>

O primeiro projeto foi uma aplicação simples feita com HTML5 para calcular a idade de uma pessoa com base na sua data de nascimento. Pense da seguinte maneira: você insere sua data de nascimento e o sistema calcula sua idade com base na data atual. Para isso, foi definido que o ano atual seria obtido utilizando o método getFullYear. A aplicação então subtrai o ano de nascimento do ano atual e exibe o resultado no console.
